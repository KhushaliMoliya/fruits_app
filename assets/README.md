# Fruits App Assets

This folder contains image assets for the Fruits App.

## Required Images

Add the following fruit images to this folder:

1. **fruit.png** - Pineapple image (for Ananas product)
   - Recommended size: 200x200 px or higher
   - Format: PNG or JPG
   
2. **fruite-item-2.jpg** - Avocado image
   - Recommended size: 200x200 px or higher
   - Format: PNG or JPG

3. **mango.png** - Mango image
   - Recommended size: 200x200 px or higher
   - Format: PNG or JPG

4. **fruite-item-4.jpg** - Passion fruit image
   - Recommended size: 200x200 px or higher
   - Format: PNG or JPG

5. **hero-img.jpg** - Pineapple image for the banner section
   - Recommended size: 200x150 px or higher
   - Format: PNG or JPG

## Image Specifications

- **Quality**: High quality fruit images
- **Background**: Transparent PNG or solid colored background
- **Style**: Natural or illustrated fruit images
- **Colors**: Vibrant colors that match the product card backgrounds

## How to Add Images

1. Find or create fruit images in PNG or JPG format
2. Place them in this `assets/` folder
3. Run `flutter pub get` to update pubspec.yaml
4. Run `flutter run` to see the images in the app

## Recommended Resources

You can find fruit images from:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)
- Or create custom illustrations

## Notes

The app currently uses placeholder icons as fallback for missing images.
Once you add the actual images, they will automatically display in the app.
